本文介绍如何在 H5 平台上使用网易会议组件 NEMeetingKit 进行语言和主题的设置。

## 设置会议语言

可以通过 [`NEMeetingLanguage`](https://doc.yunxin.163.com/meetingkit/references/web/typedoc/Latest/zh/h5/enums/NEMeetingLanguage.html) 设置会议界面的语言，支持中文、英文和日文。

**示例代码如下**：

```javascript
// 初始化会议组件

const config = {
    appKey: 'your app key',
    appName: 'your app name',
    // 或者英文NEMeetingLanguage.ENGLISH，或者日文NEMeetingLanguage.JAPANESE
    language: NEMeetingLanguage.CHINESE,
}
// 初始化会议组件
const neMeetingKit = NEMeetingKit.getInstance();
await neMeetingKit.initialize(config);
```

## 切换会议语言

您也可以通过 [`switchLanguage`](https://doc.yunxin.163.com/meetingkit/references/web/typedoc/Latest/zh/h5/interfaces/NEMeetingKit.html#switchLanguage) 方法在应用运行过程中动态切换语言。

**示例代码如下**：

```javascript
// 切换语言

const neMeetingKit = NEMeetingKit.getInstance()
neMeetingKit.switchLanguage(NEMeetingLanguage.CHINESE).then(res => {
    // 切换成功
}).catch(error => {
    // 切换失败
})
```
