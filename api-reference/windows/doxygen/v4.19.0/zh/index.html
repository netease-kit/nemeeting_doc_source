产品介绍

网易云信 NEMeetingKit 提供完善的在线会议组件，支持多人音视频，屏幕共享，成员会控操作，聊天室，白板，美颜等功能以及部分自定义功能。

  • NEMeetingKit Kit实例的创建和初始化，获取服务等方法。
  • NEMeetingService 会议服务，包含加入/创建会议，获取会议相关信息，音频订阅/取消订阅，状态监听注册等方法。
  • NEAccountService 账户服务，包含登录、登出等方法。
  • NESettingsService 配置服务，包含音视频、美颜、直播、白板、录制等的控制器，一些状态监听注册等方法。
  • NEPreMeetingService 会前服务，包含会议的预约，取消，编辑，获取列表等方法。
  • NEMeetingInviteService 会议邀请服务，包含会议邀请监听等方法。
  • NEMeetingMessageChannelService 会议消息通知服务，包含消息通知监听等方法。
  • NEContactsService 通讯录服务，包含联系人查询等方法。
  • NEFeedbackService 反馈服务，包含获取日志等，状态监听注册等方法。
