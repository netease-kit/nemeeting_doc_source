# API Reference
## 网易云信会议 SDK API 参考文档

欢迎使用网易云信会议 SDK！本仓库包含了多平台会议 SDK 的完整 API 参考文档。

### 概述

网易云信会议 SDK 提供了多平台的会议功能集成方案，支持 Android、iOS、Web、Windows、macOS 和 Linux 等多个平台。通过这些 SDK，您可以轻松地将会议功能集成到您的应用中。
