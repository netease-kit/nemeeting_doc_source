 视频编码的码率 单位为 Kbps 您可以根据场景需要，手动设置想要的码率。 - 若设置的视频码率超出合理范围，SDK 会自动按照合理区间处理码率。 - 若设置为 0，SDK将会自行计算处理。
 
NSUInteger minBitrate
