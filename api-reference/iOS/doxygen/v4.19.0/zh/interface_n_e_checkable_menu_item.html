详细描述

包含checked和uncheck两个状态的菜单项。初始时菜单项为uncheck状态，点击后可切至checked状态，如此反复。在菜单项点击回调方法中可获取当前菜单项的当前状态，并可控制菜单项是否可以进行状态迁移。
