详细描述

提供实时记录界面相关的服务接口，诸如秒录开启/停止状态监听等。 可通过 [NEMeetingKit getMeetingRealtimeRecorderService] 获取对应的服务实例。

- (void) startRealtimeRecorder: (NEMeetingRealtimeRecorderParams *) param
opts: (nullable NEMeetingRealtimeRecorderOptions *) opts
callback: (NECallback) callback 

开启实时记录，只有完成SDK的登录鉴权操作才能进入实时记录界面。 开启实时记录成功后，SDK会拉起实时记录界面页面，调用方不用做其他操作

参数
param实时记录界面参数对象，不能为空
opts实时记录界面选项对象，可空；当未指定时，会使用默认的选项
callback回调接口。该回调不会返回额外的结果数据
