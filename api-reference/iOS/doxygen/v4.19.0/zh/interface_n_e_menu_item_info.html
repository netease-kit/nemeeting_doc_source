菜单项图标资源名称，如果为自定义注入菜单，需指定真实存在的图标资源名称；如果为SDK内置菜单，可指定为"0"，由SDK加载内置图标。图标资源尺寸大小规格为：1x: 24px * 24px ; 2x: 48px * 48px，以此类推。
