详细描述

菜单项基类对象。菜单项由 itemId 唯一标识，包含标题和图标，并可控制对不同参会者角色是否可见。 目前SDK提供了单状态菜单项 NESingleStateMenuItem 、双状态菜单项 NECheckableMenuItem 实现可供使用。单状态菜单始终展示相同的标题与图标；多状态的菜单包含与状态数一一对应的标题与图标，在菜单状态变更时会触发UI更新。通过注册回调可监听自定义注入菜单的点击事件(SDK内置菜单不会触发回调)。
