BOOL enableAudioShare
 

详细描述

提供屏幕共享时必要的配置信息和选项开关，如音频共享开关

属性说明

◆ enableAudioShare

- (BOOL) enableAudioShare
readwritenonatomicassign

开启/关闭音频共享功能。 开启后，在发起屏幕共享时，会同时自动开启设备的音频共享； 关闭后，在发起屏幕共享时，不会自动打开音频共享，但可以通过UI手动开启音频共享。 该设置默认为关闭。
