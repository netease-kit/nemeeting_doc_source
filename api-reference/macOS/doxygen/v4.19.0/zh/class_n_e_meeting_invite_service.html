const NEEmptyCallback & callback )
pure virtual

接受邀请，加入一个当前正在进行中的会议

加入会议成功后，SDK会拉起会议页面，调用方不用做其他操作。

参数
param会议参数对象，不能为空
opts会议选项对象，可空；当未指定时，会使用默认的选项
callback回调接口。该回调不会返回额外的结果数据
