## 自定义会中 UI 单击监听

您可以通过实现 [`NEMeetingViewClickListener`](https://doc.yunxin.163.com/meetingkit/references/web/typedoc/Latest/zh/types/NEMeetingViewClickListener.html) 接口来自定义会议信息的显示和管理参会者中的邀请功能。

**示例代码如下：**

```javascript
// 创建会议视图单击监听器

const neMeetingKit = NEMeetingKit.getInstance();
const meetingService = neMeetingKit.getMeetingService();

const viewClickListener = {
    onClickInviteIcon: (meetingInfo) => {
        // 成员列表邀请点击事件回调
        // 处理点击逻辑
        // 如果要拦截默认逻辑则返回true
        return false
    },
    onClickMeetingInfoIcon: (meetingInfo) => {
        // 会议信息icon击事件回调
        // 处理点击逻辑
        // 如果要拦截默认逻辑则返回true
        return false
    }
}

// 设置会议视图单击监听器
meetingService.setViewClickListener(viewClickListener);
```

## 其他 UI 定制选项

NEMeetingKit 中的 [`NEMeetingOptions`](https://doc.yunxin.163.com/meetingkit/references/web/typedoc/Latest/zh/types/NEMeetingOptions.html) 提供了丰富的会中 UI 配置选项，以下是一些常用配置：

```javascript
const neMeetingKit = NEMeetingKit.getInstance();
const meetingService = neMeetingKit.getMeetingService();
const params = {
    meetingNum: '您的会议号',
    displayName: '您的会议昵称',
    password: '您的会议密码',
}
// 创建会议选项对象
const options = {
    // 隐藏聊天功能
    noChat: true,
    // 隐藏邀请按钮
    noInvite: true,
    // 隐藏白板共享
    noWhiteBoard: true,
    // 配置视频画面上显示用户名
    showNameInVideo: true,
    // 配置会议信息显示选项
    meetingIdDisplayOption: NEMeetingIdDisplayOption.DISPLAY_ALL,
    // 配置表情回应是否显示
    showEmojiResponse: true,
    // 配置举手功能是否显示
    showHandsUp: true,
    // 配置会议信息查看入口是否显示
    showMeetingInfo: true,
    // 配置会议剩余时间提醒是否显示
    showMeetingRemainingTip: true,
    // 配置摄像头镜像
    enableVideoMirroring: true,
}

// 使用配置项加入或开始会议
meetingService.joinMeeting(params, options).then(() => {
    // 加入会议成功
}).catch((error) => {
    // 加入会议失败
});

// 或开始会议
meetingService.startMeeting(params, options).then(() => {
    // 创建会议成功
}).catch((error) => {
    // 创建会议失败
});
```
