## 自定义会中 UI 单击监听

您可以通过实现 [`NEMeetingViewClickListener`](https://doc.yunxin.163.com/docs/interface/meetingkit/windows/doxygen/Latest/zh/class_n_e_meeting_view_click_listener.html) 接口来自定义会议信息的显示和管理参会者中的邀请功能。

**示例代码**：

```C++
// 创建会议视图单击监听器
auto svc = NEMeetingKit::getInstance()->getMeetingService();
svc->setViewClickListener(this);
// 设置会议视图单击监听器，需要在加入会议前设置
NEMeetingKit.getInstance().setMeetingViewClickListener(viewClickListener);

void HawkManager::onClickMeetingInfoIcon(const NEMeetingInfo& meetingInfo) {

void HawkManager::onClickInviteIcon(const NEMeetingInfo& meetingInfo) {
}
```

## 其他 UI 定制选项

NEMeetingKit 中的 [`NEMeetingOptions`](https://doc.yunxin.163.com/docs/interface/meetingkit/windows/doxygen/Latest/zh/class_n_e_meeting_options.html) 提供了丰富的会中 UI 配置选项，以下是一些常用配置：

```C++
// 创建会议选项对象
NEJoinMeetingParams params;
NEJoinMeetingOptions options;

// 隐藏聊天功能
options.noChat = true;

// 隐藏邀请按钮
options.noInvite = true;

// 隐藏白板共享
options.noWhiteBoard = true;

// 配置视频画面上显示用户名
options.showNameInVideo = true;

// 配置会议信息显示选项
options.meetingIdDisplayOption = NEShowMeetingIdOption::kDisplayAll;

// 配置表情回应是否显示
options.showEmojiResponse = true;

// 配置举手功能是否显示
options.showHandsUp = true;

// 配置会议中是否显示会议信息查看入口
options.showMeetingInfo = true;

// 配置是否显示会议剩余时间提醒
options.showMeetingRemainingTip = true;

// 配置摄像头镜像
options.enableFrontCameraMirror = true;

// 使用配置项加入或开始会议
svc->joinMeeting(params, options, callback);
// 或
svc->startMeeting(params, options, callback);
```
