本文介绍如何在 Linux 平台上使用网易会议组件 NEMeetingKit 进行语言和主题的设置。

## 设置会议语言

可以通过 [`NEMeetingLanguage`](https://doc.yunxin.163.com/docs/interface/meetingkit/linux/doxygen/Latest/zh/kit__define__meeting_8h.html#a2ff3dbbfa2bfc7a2bb3dca4748eb654a) 设置会议界面的语言，支持中文、英文和日文。

**示例代码如下：**

```C++
// 创建会议 SDK 配置
nem_sdk_interface::NEMeetingKitConfig config;

// 设置语言类型（注意：必须在初始化 SDK 时设置）
// 中文
config.setLanguage(NEMeetingLanguage::kNEChinese);

// 或者英文
// config.setLanguage(NEMeetingLanguage::kNEEnglish);

// 或者日文
// config.setLanguage(NEMeetingLanguage::kNEJapanese);

// 或者根据系统语言自动选择
// config.setLanguage(NEMeetingLanguage::kNEAutomatic);

// 初始化会议 SDK
NEMeetingKit::getInstance()->initialize(config, [this](
                                                        MeetingErrorCode errorCode, const std::string& errorMessage, const NEMeetingCorpInfo& info) {});
```

## 切换会议语言

您也可以通过 [`switchLanguage`](https://doc.yunxin.163.com/docs/interface/meetingkit/linux/doxygen/Latest/zh/class_n_e_meeting_kit.html#af1ae0b4937e24f9aa9f2247bb88a73b0) 方法在应用运行过程中动态切换语言。

**示例代码如下：**

```C++
NEMeetingKit::getInstance()->switchLanguage((NEMeetingLanguage)type, [this](MeetingErrorCode errorCode, const std::string& errorMessage) {
        PrintLog("MeetingSDK switchLanguage code:" + std::to_string(errorCode) + ", errorMessage: " + errorMessage);
    });
```

## 切换主题色

您可以通过 [`NEMeetingThemeType`](https://doc.yunxin.163.com/docs/interface/meetingkit/linux/doxygen/Latest/zh/kit__define__meeting_8h.html#a435c88486b665725798279a74a512789) 根据需要设置会议界面的主题，支持深色主题、浅色主题以及跟随系统切换。

**示例代码如下：**

```C++
// 设置主题类型
// 浅色主题
NEMeetingThemeType themeType = NEMeetingThemeType::kNEMeetingThemeTypeLight;

// 或者深色主题
// NEMeetingThemeType themeType = NEMeetingThemeType::kNEMeetingThemeTypeDark;

// 或者跟随系统
// NEMeetingThemeType themeType = NEMeetingThemeType::kNEMeetingThemeTypeSystem;

// 切换会议主题
sdk->switchTheme(themeType, resp);
```
