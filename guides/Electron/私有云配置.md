本文介绍如何配置会议组件的私有云服务。

## 前提条件

请在您的企业内部部署企业专用的会议服务器，以供会议组件 NEMeetingKit 连接。

:::note note
该部署过程需要您与网易云信服务团队协作完成，因此若有私有云服务相关需求，请先前往 [云信官网](https://yunxin.163.com/) 联系商务经理一对一沟通您的需求。
:::


## 方式一：通过 serverUrl 设置（推荐）

调用 `initialize` 方法初始化 NEMeetingKit 时，配置 `NEMeetingKitConfig` 的 `serverUrl` 参数为私有化服务器地址。

**示例代码如下:**

```TypeScript
const neMeetingKit = NEMeetingKit.getInstance();

const config = {
    appKey: 'Your_App_Key',
    appName: 'Your_App_Name',
    serverUrl: 'Your_Server_Url', //配置私有化服务器地址
}
neMeetingKit.initialize(config);
```

## 方式二：通过配置参数设置

私有云服务器部署完成后，网易云信会提供给您对应的会议私有云配置文件。您在初始化会议组件 SDK 时，需要进行相关的配置，以便 SDK 能接入到目标私有云服务器上。

调用 `initialize` 方法初始化 NEMeetingKit 时，配置 `config` 参数的 `serverConfig` 字段。

**示例代码如下**：

···

    httpsEnabled: boolean;

    nosLbs: string;

/**

    mediaProxyServer?: string;

 * 云代理功能配置的服务器地址
 */
