<!--- keywords: 美颜,会议组件 -->

NEMeetingKit 提供云信自研的基础美颜功能，帮助用户在音视频通话会议场景中，对人脸进行美肤等美颜调整。

若需要开启美颜功能，请联系技术支持为您进行开通，并根据以下示例进行配置。

```javascript
const neMeetingKit = NEMeetingKit.getInstance();
const settingsService = neMeetingKit.getSettingsService();
// 登录后，查选美颜功能是否支持
const isBeautyFaceSupported = await settingsService.isBeautyFaceSupported();

/**
 * 设置美颜参数
 *
 * @param value 传入美颜等级，参数规则为[0,10]整数
 */
await settingsService.setBeautyFaceValue(1);

/** 获取当前美颜参数，关闭返回0 */
const beautyFaceValue = await settingsService.getBeautyFaceValue();
```
