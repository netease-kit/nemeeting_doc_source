本文详细介绍如何在Linux平台上使用网易会议组件 NEMeetingKit 配置会议水印功能。会议水印功能在会议画面上显示用户身份信息，有效防止会议内容被截屏后泄露。

## 前提条件

开始本文操作前，请确保您已经完成了 [登录鉴权](https://doc.yunxin.163.com/meeting/guide/DI1MTY4Njc?platform=linux) 的相关配置。

## 第一步：创建水印配置对象

创建 [`NEWatermarkConfig`](https://doc.yunxin.163.com/docs/interface/meetingkit/linux/doxygen/Latest/zh/struct_n_e_watermark_config.html) 水印配置对象，提供水印中可能需要显示的用户信息。

```C++
//创建水印配置对象，包含用户身份信息
NEWatermarkConfig watermarkConfig;
watermarkConfig.name =  "张三";          // 用户姓名
watermarkConfig.phone =  "13812345678";   // 手机号码
watermarkConfig.email =  "zhangsan@example.com";          // 邮箱
watermarkConfig.jobNumber =  "EMP10086";          // 工号
```

## 第二步：配置入会参数并显示水印

将水印配置对象设置到会议参数中，并使用该配置入会。系统默认会使用 `{name}` 格式显示水印，即使用 `NEWatermarkConfig.name` 字段的值作为水印内容。

```C++
// 在会议参数中设置水印配置
NEStartMeetingParams params;
params.displayName = "张三";
params.watermarkConfig = watermarkConfig;

// 使用设置好的参数创建会议
svc->startMeeting(params, options, callback);
```

## 第三步：自定义水印样式（可选）

如需使用其他格式的水印（如 `{name}+{phone}` 或其他组合），请 [提交工单](https://app.yunxin.163.com/global/service/ticket/create) 联系网易云信技术支持工程师进行配置。

技术支持工程师将根据您的需求配置自定义水印格式。配置完成后，您仍使用相同的代码方式设置水印，但系统将按照您定制的格式显示水印内容。

## 第四步：开启或关闭会中水印

配置水印对象并传入后，主持人或联席主持人可在会议中 **更多** > **安全** > **会议水印** 中开启或关闭水印。开启后，所有参会者的画面上将显示已配置的水印。
