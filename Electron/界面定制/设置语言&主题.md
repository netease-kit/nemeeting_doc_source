本文介绍如何在 Electron 框架上使用网易会议组件 NEMeetingKit 进行语言和主题的设置。

## 设置会议语言

可以通过 [`NEMeetingLanguage`](https://doc.yunxin.163.com/meetingkit/references/web/typedoc/Latest/zh/electron/enums/NEMeetingLanguage.html) 设置会议界面的语言，支持中文、英文和日文。

**示例代码**：

```TypeScript
// 初始化会议组件
import NEMeetingKit  from 'nemeeting-electron-sdk';

const config = {
    appKey: 'your app key',
    appName: 'your app name',
    language: NEMeetingLanguage.CHINESE,
}
// 初始化会议组件
const neMeetingKit = NEMeetingKit.getInstance();
await neMeetingKit.initialize(config);
```

## 切换会议语言

您也可以通过 [`switchLanguage`](https://doc.yunxin.163.com/meetingkit/references/web/typedoc/Latest/zh/electron/interfaces/NEMeetingKit.html#switchLanguage) 方法在应用运行过程中动态切换语言。

**示例代码**：

```TypeScript
// 切换语言
import NEMeetingKit  from 'nemeeting-electron-sdk';

const neMeetingKit = NEMeetingKit.getInstance();
await neMeetingKit.switchLanguage(NEMeetingLanguage.CHINESE);
```

## 切换主题色

您可以通过 [`switchTheme`](https://doc.yunxin.163.com/meetingkit/references/web/typedoc/Latest/zh/electron/interfaces/NEMeetingKit.html#switchTheme) 根据需要设置会议界面的主题，支持深色主题、浅色主题以及跟随系统切换。

**示例代码**：

```TypeScript
// 切换主题
import NEMeetingKit  from 'nemeeting-electron-sdk';

const neMeetingKit = NEMeetingKit.getInstance();
await neMeetingKit.switchTheme(NEMeetingThemeType.LIGHT);
```
